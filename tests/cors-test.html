CORS Troubleshooting:

  • If you see a CORS error, the backend server needs to include the appropriate CORS headers.
  • The backend should include: Access-Control-Allow-Origin: * (or specific origin)
  • For preflight requests, the backend should also handle OPTIONS requests.
  • Check that both servers are running (frontend on port 8080, backend on port 3000).
